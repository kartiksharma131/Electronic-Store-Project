# Electronic-Store-Project
This is a major backend project to manage a electronic store using Java , SpringBoot and related technologies

# Tech Stack
    - Java
    - SpringBoot
    - MySQL
    - JWT
    - Swagger
    - Hibernate
    - Spring Security
    - ReactJS

# Services/Modules
    - User Management Service
    - Product Management Service
    - Category Management Service
        - Sub Category Management Service(IF SCALE INCREASES)
    - Order Management Service
    - Payment Management Service
    - Cart Management Service
    - User Role Management Service
    - Other Services
